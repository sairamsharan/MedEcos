# MedEcos Frontend

The Frontend of the MedEcos ecosystem consists of three distinct client applications designed for specific roles within the medical platform.

## Sub-Applications

### 1. Patient App (`med_ecos_patient`)
The interface for patients to manage their profile, view medical records, and interact with doctors and pharmacies.

### 2. Doctor App (`med_ecos_doctor`)
The interface for doctors to manage appointments, view patient records, and safely issue prescriptions.

### 3. Pharmacist App (`med_ecos_pharmacist`)
The interface for pharmacists to verify prescriptions, manage inventory, and seamlessly connect with patients.

## Getting Started

Each application is a separate project. Navigate to the respective folder to run or build the application.

```bash
# Example: Running the Patient App
cd med_ecos_patient
npm install
npm run dev
```
*(Please adapt commands based on the specific framework used within the sub-directories, such as React/Vue/Svelte)*
