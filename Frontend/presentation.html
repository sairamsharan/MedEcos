The Problem

India's Healthcare is Fragmented 💔

📁

Physical Paper Files

Patients carry bulky physical files to every appointment. Records get lost, damaged, or forgotten — making continuity of care nearly impossible.

👨‍⚕️

Doctors Lack History

A doctor meeting a new patient has zero history. Dangerous drug combinations are prescribed unknowingly due to lack of previous prescription data.

💊

Manual Prescription Verification

Pharmacists manually read handwritten prescriptions, causing errors, delays, and potential drug dispensing mistakes that impact patient safety.

🔬

No Digital Lab Workflow

Lab technicians receive handwritten test orders. There's no way for patients to track test status or receive digital results in real time.

Under the Hood

Technology Stack ⚙️

📱

Flutter (Dart)

All 4 stakeholder apps built in Flutter. Single codebase deploys to Android, iOS, Web, and Windows Desktop.

why: 1 codebase = 4 platforms
🔥

Firebase / Node.js Backend

Firebase Auth + Firestore for real-time data sync. Node.js + Express for custom APIs. WebSocket for live lab status updates.

why: real-time + scalable
🤖

Google Gemini AI API

Powers the drug interaction detection engine. Caches known drug pairs locally; Gemini is called only for unknown combinations to minimize API usage.

why: smart + cost-efficient
🍃

MongoDB + Geospatial Index

Doctor profiles stored with GeoJSON Point locations. MongoDB's $nearSphere query returns nearby doctors sorted by distance in milliseconds.

why: location queries at scale
📲

QR Code + ABHA API

Patient ABHA ID is encoded as a QR. Integration with Ayushman Bharat Digital Health Mission APIs for official ID verification and record linkage.

why: govt. interoperability
🗺️

flutter_map + OpenStreetMap

Interactive map with doctor location pins, zero API key cost. Geolocator package for GPS. Custom pin widgets indicate availability in real time.

why: cost-free, open-source
