# MedEcos

MedEcos is a comprehensive Medical Ecosystem that connects Patients, Doctors, and Pharmacists into a unified platform.

## Project Structure

This project is divided into two main components:

- **[Backend](./Backend/README.md)**: The central Node.js REST API that powers the ecosystem, manages authentication, and handles data securely.
- **[Frontend](./Frontend/README.md)**: Contains three distinct client applications tailored for different user roles (Patient, Doctor, Pharmacist).

## Getting Started

To get started with development or deployment, please refer to the specific documentation for each component:

1. [Backend Documentation](./Backend/README.md)
2. [Frontend Documentation](./Frontend/README.md)

## Overall Features
* Secure Authentication and Role-based Access Control
* Patient Health Record Management
* Doctor Prescription and Consultation System
* Pharmacist Verification and Dispensing Workflow
